# Project 03 Resume

## Description
A personal resume website built with Next.js App Router and Tailwind CSS.

## How to Run Locally

1. npm install
2. npm run dev
3. Open http://localhost:3000 in your browser

## Features Implemented

- Professional Summary
- Education Section
- Work Experience Section
- Technical Projects Section
- Skills Section
- Tailwind CSS Styling
- Semantic HTML Structure
